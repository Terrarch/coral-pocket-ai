## 📖 Introduction
Built entirely on a Single-File architecture. It perfectly aggregates tier-one models like ChatGPT (GPT-5.3), Claude-opus-4.1, Gemini-3.1-pro-preview, DeepSeek, and Grok.

## ✨ Highlights
- 🌈 **Multi-Model Support**: Switch seamlessly between the latest GPT, Claude, Gemini, and DeepSeek endpoints.
- 📊 **Billing Console**: [🔴 Access the Console Here](https://api.inyu.top/console) to manage your API balance, tokens, and billing.
- 📱 **Mobile-Native PWA**: Installable directly to your home screen.
- 📎 **Multimodal Ready**: Drag-and-drop support for images and document attachments.
