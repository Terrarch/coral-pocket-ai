## 📖 简介

**珊瑚口袋 AI** 颠覆了传统 AI 客户端的臃肿与商业化套壳限制。它基于 `Single-File` (纯单文件) 架构，**没有后端，无需安装，下载仅需几秒**。

我们用最纯净的 HTML/JS 实现了当今最硬核的功能：无论是寻找免梯子稳定直连的方法，还是规避官方风控，本客户端都能完美聚合 **ChatGPT (GPT-5.3)、Gemini-3.1-pro-preview、Claude-opus-4.1、DeepSeek、Grok** 等最新大厂模型。独创“打字机流式双缓冲引擎”，为您提供如丝般顺滑的跨平台对话体验。您的每一条聊天记录、每一个 API 密钥，均 **100% 留存在本地设备**，隐私绝对安全。

## ✨ 核心亮点

- ⚡ **万能兼容接口**：深度遵循 `OpenAI /v1/chat/completions` API 标准规范。支持自由重写 Base URL，完美适配各类高性价比的第三方中转代理与高速分发平台。
- 🌈 **全网模型自由切**：右上角智能下拉框，一键无缝切换所有前沿多模态视觉与大语言模型。
- 📊 **额度控制台**：侧边栏/顶部内置「[🔴 API 独立总控看板](https://api.inyu.top/console)」直达入口。随时随地可视化分析您的备用接口账单、Token 余额与轮询速率限制。
- 📱 **移动端原生体验**：专为单手操作优化。支持 PWA 一键安装至手机桌面，享受 iOS 级顺滑全屏免广告体验。
- 🌊 **极致丝滑打字机**：自研抗网络抖动流式引擎，即使在长文本代理弱网环境下，也能模拟出人类书写般的均匀吐字节奏。
- 📎 **全能多模态**：支持拖拽上传图片与文档附件，独立视觉沙盒，完美联动多模态视觉模型。
- 🔒 **无痕隐私保护**：100% 去中心化架构。无任何中间服务器偷窥您的数据集。

## 🚀 极速部署与使用

### 方式一：在线免流体验
👉 [直接访问我们的云端体验地址](https://hua.yooyo.asia/html_files/20260225_021715_26766a.html)，打开即用，断网亦可查阅历史。

### 方式二：本地私有化部署 (推荐安全做法)
1. 下载本仓库中的 `index.html`。
2. 双击在任意现代浏览器 (Chrome/Edge/Safari) 中打开。
3. 点击左下角 **⚙️ 接口配置**，填入您的 API Base URL 和 Key 即可开始无界对话。

> 💡 **寻觅海外节点遇到阻碍？**
> 对于由于信用卡风控报错、无法注册体验 GPT-5.3 或 Claude-4.1 完全体的用户，您可在内置的 **[API 标准前端中台 (Console)](https://api.inyu.top/console)** 轻松获取国内免封禁直连的极速中转密钥，享受即插即用体验。
